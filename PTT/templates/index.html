What is it?

A lightweight flask web application for using and tuning OpenAI's GPT-3, Codex and DALL-E data models.

How to use ?

Select from the Navigation bar which data model you want to use. Details for each are explained in their respected pages.

Which model should I use?

See description below for a quick discription of each

GPT-3

For natrual language tasks. GPT-3 is strong at text completion. Given a prompt it will return an answer that follows the prompts context or pattern.

Codex

For code completion tasks. Suited for generating snippets of code across many different languages. Code may not be vaild.

Dalle-2

For image generation tasks.
